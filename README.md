Temperature Converter
A lightweight iOS app that converts Celsius to Fahrenheit in real-time using MVVM architecture and Combine.

Features
⚡️ Real-time Conversion: Updates as you type.

🔄 Reset: One-tap to clear all fields.

📱 Modern UI: Built programmatically with Auto Layout and StackViews.

Tech Stack
Swift / UIKit (Programmatic UI)

MVVM Architecture

Combine (Reactive data binding)
